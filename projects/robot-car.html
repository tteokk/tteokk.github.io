Robot Car

This robot car project was an exercise in low-level embedded programming, where I implemented line-following behavior using C and hexadecimal instructions. The car used basic sensor input and motor control logic to navigate a course autonomously.

Embedded Systems Focus

  • Programmed microcontroller in C with direct register manipulation and hexadecimal opcodes.
  • Used analog sensors to detect black tape lines and route-following edges.
  • Controlled motors using basic PWM and conditional logic based on sensor thresholds.
  • Created a simple instruction parser using lookup tables to trigger behavior by sensor states.
  • Debugged behavior using onboard LEDs and step-wise code execution during lab testing.
