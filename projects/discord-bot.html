Discord Bots

A collection of custom Discord bots built for personal use and small communities. These bots handled everything from motivational quotes to Pomodoro timers and friendly exam betting games among friends, helping automate fun and productivity features on Discord servers.

Software Development Focus

  • Developed bots in Python using the Discord.py API and deployed locally or on cloud services.
  • Created event-driven commands and responses triggered by user input and server events.
  • Built a Quote Bot that sent random quotes paired with random idol images.
  • Designed a Pomodoro Timer Bot that accepted time input and played a song on session end.
  • Implemented an Exam Bet Bot that tracked users' score guesses and calculated winner statistics over time.
