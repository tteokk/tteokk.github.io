2 Axis Limit Switch

This embedded systems project controls two motors using a microcontroller (STM32F401) and a pair of limit switches. The system uses analog-to-digital conversion (ADC) to read input signals and uses GPIO polling interrupts to stop motors when they hit mechanical limits.

Overview

  • Controlled 2-axis movement with position-limiting interrupts for safety and precision.
  • Used STM32 microcontroller and custom C firmware to handle real-time motor response.
  • Verified limit switch triggering and safe stopping through GPIO state checks.

Embedded Systems Focus

  • Configured STM32 GPIO pins to detect limit switch triggers with pull-down resistors.
  • Used ADC channels to read analog voltage levels for position sensing.
  • Implemented interrupt-based polling logic to stop motor PWM output when limits were hit.
  • Wrote state machine logic to move, stop, and reset the motor positions safely.
  • Tested using UART output and oscilloscope to verify GPIO signal timing and correctness.
