GLCD Game

A custom game built on an STM32 development board using a graphical LCD (GLCD), joystick, LEDs, and push buttons. This project focused on real-time embedded programming, concurrency with threads and mutexes, and low-level display control in C.

Embedded Systems & Concurrency

  • Programmed game logic in C using STM32 microcontroller with direct register access.
  • Interfaced with a KS0108-based GLCD using parallel data bus and control signals.
  • Handled real-time joystick inputs using ADC and used push buttons for in-game actions.
  • Implemented multi-threaded control using RTOS-style tasks and synchronization via mutexes.
  • Used structs, pointers, and memory-mapped I/O to efficiently manage frame drawing and movement.
  • Debugged concurrency issues like shared data race conditions using LED state indicators.
