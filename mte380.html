MTE 380: Autonomous Line Following Robot

Our MTE 380 project involved designing and building a two-wheeled autonomous robot capable of navigating a line-based course and performing tasks such as object pickup and drop-off using real-time sensor feedback and controls. The robot was built using an STM32 microcontroller and optimized for speed, stability, and task accuracy.

Project Overview

  • Used STM32F401 Nucleo-64 for real-time motor and sensor control.
  • Implemented polling and interrupt-based sensor detection for object and line tracking.
  • PID control was used for both steering and task execution timing.
  • Robot performed autonomous figure detection, pickup, navigation, and drop-off.
